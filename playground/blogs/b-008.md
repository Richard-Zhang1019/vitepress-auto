# console.log 的花里胡哨的用法

> 2022-11-29 23:00:36

在去大理的高铁上，手机也快没电了实在是无聊掏出来电脑写一会

昨天在群里看见群友发的一个骚操作，console.log 打印的结果是带着彩虹背景图，感觉很~~花里胡哨~~有意思，然后就去查了查，有这些设定

- %s: for string
- %i or %d: for integers
- %f: for float numbers
- %o: for DOM element
- %O: for JS object
- %c : for adding styles to your log

比如我们要加个样式就可以这样

```js
console.log('%cHelloWorld', 'color: red; font-size: 30px')
```
